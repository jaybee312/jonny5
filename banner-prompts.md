# askjonny5.com — banner prompts (built for GPT image gen)

GPT can't handle a complex art brief, so each prompt is stripped to one thing it
can actually draw: one character, one short line, one simple scene. No negative
lists, no nameplates, no footer text baked in — that text gets garbled, and it's
cleaner as HTML on the site anyway.

## How to run these
1. Generate ONE at a time. Don't ask for all four in one go.
2. Upload that persona's existing strip as a reference image first, so GPT keeps
   the same face. Then paste the prompt.
3. The speech bubble is the ONLY text in the image. Keep it short.
4. If the character drifts, regenerate with the reference attached and add:
   "keep this exact same character."
5. Skip nameplates, the bottom brand strip, whiteboard text, mug text — we'll add
   any labels as clean HTML on the site.
