# askjonny5.com — campaign links

These exist only for your convenience when posting. Google Analytics reads the
UTM tags at click time, so nothing here needs to be "live." Keep this file so you
reuse the exact same labels every time — consistent labels = clean reports.

## The pattern

    https://askjonny5.com/?utm_source=PLATFORM&utm_medium=social&utm_campaign=CAMPAIGN#TAB

- PLATFORM — where you posted: reddit, linkedin, substack, x …
- CAMPAIGN — which persona (keep it lowercase, exactly as written below)
- TAB — the tab the link opens (the part after #)

## Persona → tab → campaign

    Market Outlook   #outlook      market_outlook     (macro / market post)
    Pete (Trucker)   #trucker      pete_trucker       (comic)
    Lisa (Fleet)     #fleet        lisa_fleet         (comic)
    Maria (Rest.)    #restaurant   maria_restaurant   (comic)
    Sam (E-Comm)     #ecomm        sam_ecomm          (comic)
    Origin (About)   #about        origin_about       (origin / credibility post)
